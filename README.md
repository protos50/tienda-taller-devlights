# Tienda Taller 70s

Mini e-commerce hecho en Next.js con tema retro y carrito global persistente.

![Preview del sitio](./public/Captura%20de%20pantalla%202025-12-14%20185257.png)

## ¿Cómo lo corro?

```bash
npm install
npm run dev
```

Visita <http://localhost:3000> para ver la tienda. El carrito guarda el estado en `localStorage`, así que podés refrescar sin perder tus productos.
